üéÆ How to Play

You are a programmer trapped in a corrupted digital maze. Use your Java knowledge to debug your way out!

üïπÔ∏è Game Controls

  • Desktop: Use arrow keys to move
  • Mouse: Left-click or right-click anywhere on the maze

üéØ Game Mechanics

  • Questions: Each move triggers a Java programming question
  • Correct Answer: Move forward and earn +10 coins
  • Wrong Answer: Return to start, lose 5 coins (or spend 5 coins to stay)
  • Coins: Collect data tokens throughout the maze (+5 coins each)
  • Dead Ends: Cost 2 coins as debugging penalty
